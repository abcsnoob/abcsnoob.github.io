Hướng dẫn cài Datapack vào Server Minecraft

Datapack là tập hợp các dữ liệu tùy chỉnh giúp thêm tính năng mới cho server Minecraft của bạn mà không cần mod.

Bước 1: Tải datapack

Tải file datapack dưới dạng .zip hoặc thư mục chứa các tệp datapack.

Bước 2: Truy cập thư mục server

Mở thư mục chứa server Minecraft của bạn trên máy tính hoặc hosting.

Bước 3: Vào thư mục world

Đi đến thư mục chứa thế giới (world) của server, ví dụ:

./world/

Bước 4: Vào thư mục datapacks

Mở thư mục datapacks nằm trong thư mục thế giới:

./world/datapacks/

Bước 5: Thêm datapack vào

Copy file datapack (file .zip hoặc thư mục datapack) vào thư mục datapacks.

Bước 6: Khởi động lại server

Restart lại server Minecraft để datapack được kích hoạt.

Bước 7: Kiểm tra datapack

Vào server và gõ lệnh sau để kiểm tra danh sách datapack đã được tải:

/datapack list

Lưu ý:

  • Đảm bảo datapack tương thích với phiên bản Minecraft server của bạn.
  • Nên sao lưu dữ liệu server trước khi cài đặt datapack mới.
